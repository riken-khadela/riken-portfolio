This is a public web app built using Flask framework that allows users to purchase and use royalty images. The app has various payment options including debit/credit cards, UPI, and PayPal to facilitate transactions. The project also has a chat feature to improve communication and transactions between buyers and sellers. Although many requirements were added, the project has not been completed yet.

What is about this project

Welcome to my latest project - a public web app built on Flask. This project is designed to offer a platform that provides features similar to Pinterest, but with a unique twist. Unlike Pinterest, this project requires users to pay a royalty fee in order to use the images on the platform. This app is built to cater to the need for a secure and reliable platform for buying and selling royalty images.

Challanges We Have Faced

One of the major challenges of this project was integrating payment systems. To ensure the smooth functioning of the platform, it was necessary to provide various payment options such as debit, credit card, UPI, and PayPal. This required an in-depth understanding of different payment gateways and ensuring seamless integration with the platform. Another challenge was providing a secure and user-friendly chat system for communication between buyers and sellers. This required careful planning and execution to make sure the system was efficient and easy to use.
